Learning Git and GitHub

Introduction

Git is a distributed version control system that allows developers to track changes in their codebase. GitHub is a platform built on top of Git that provides a collaborative environment for developers to share and manage code.

Why Use Git & GitHub?

  • Version Control: Keep track of every change made to your code.
  • Collaboration: Work with other developers and manage contributions.
  • Backup: Your code is safely stored and can be accessed from anywhere.
  • Integration: Integrate with various CI/CD tools for automated testing and deployment.

Basic Git Commands

Command Description
git init Initialize a new Git repository
git clone [url] Clone an existing repository
git add [file] Stage changes for commit
git commit -m "message" Commit changes with a message
git push Push changes to a remote repository
git pull Fetch and merge changes from a remote repository
